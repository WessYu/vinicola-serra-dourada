# Vinícola Serra Dourada 

Projeto de portfólio: landing page de vinícola com vitrine de produtos, filtro/ordenação, carrinho (drawer) com persistência via localStorage e modal de agendamento.

## Stack
- React + Vite
- TailwindCSS

## Rodar localmente
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
